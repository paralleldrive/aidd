## Glassmorphism

The signature aesthetic of our design system combines frosted glass effects with subtle transparency.

### Glass Surface

Our `.glass-surface` class creates the iconic frosted glass effect:

```css
.glass-surface {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
}
```

## Spacing Scale

Consistent spacing creates visual rhythm and improves readability.

| Token | Value | Usage |
|-------|-------|-------|
| `0.5rem` | 8px | Minimal gaps, icon spacing |
| `0.75rem` | 12px | Small component gaps |
| `1rem` | 16px | Default spacing |
| `1.25rem` | 20px | Component padding |
| `1.5rem` | 24px | Section spacing |
| `2rem` | 32px | Large gaps, margins |
| `3rem` | 48px | Section separators |
| `4rem` | 64px | Page-level spacing |

## Accessibility

Our design system meets WCAG 2.1 Level AAA standards for color contrast.

### Contrast Ratios

- **Text Primary (#e0e0ff) on Dark (#0a0a1f)**: 12.5:1 ✓ AAA
- **Text Secondary (#a0a0c0) on Dark (#0a0a1f)**: 7.2:1 ✓ AAA
- **Neon Cyan (#00d9ff) on Dark (#0a0a1f)**: 9.8:1 ✓ AAA
- **High Priority (#ff5252) on Dark (#0a0a1f)**: 5.1:1 ✓ AA Large

### Touch Targets

All interactive elements meet the **44px × 44px minimum touch target** size for mobile accessibility (WCAG 2.5.5).

### Focus Indicators

Keyboard focus is clearly indicated with neon cyan outlines:

```css
:focus-visible {
  outline: 2px solid #00d9ff;
  outline-offset: 2px;
}
```

---

## Motion Design

Subtle animations enhance user experience without overwhelming.

### Transition Speeds

- **Fast**: 0.2s - Hover states, button presses
- **Medium**: 0.3s - Modal open/close, expansions
- **Slow**: 0.5s - Page transitions, complex animations

### Easing Functions

- **ease-out**: Default for most interactions
- **ease-in-out**: For smooth bidirectional animations
- **cubic-bezier(0.34, 1.56, 0.64, 1)**: Playful bounce effect
