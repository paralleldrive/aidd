# ECS best practices

When working with `@adobe/data/ecs`, read the **ECS (Entity Component System)** section of the canonical agent guide:

- **Package consumers:** `node_modules/@adobe/data/AGENTS.md`
- **Package editors:** `packages/data/AGENTS.md` (in this repo)

That guide covers plugin property order, composition, Store API, actions, naming conventions, and type utilities. Follow it when creating or modifying `Database.Plugin` definitions.
