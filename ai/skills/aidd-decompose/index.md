### 🧩 aidd-decompose

**File:** `SKILL.md`

Decompose a large PR into smaller, reviewable breakout PRs in dependency order. Use when a PR exceeds ~2000 lines, when review feedback requests smaller chunks, or when you need to track and sync feedback across breakout PRs in progress.
