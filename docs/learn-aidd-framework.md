# Learn aidd Framework

There's a lot to learn about aidd Framework. It is more than a framework. It is our attempt to share the insights about software engineering that really move the needle in terms of planning, building, deploying, and validating production-quality software with a velocity and scale that was never possible before AI agents.

There are a few different areas you need to learn to get good at aidd Framework:

- Product Management
- User Story Mapping
- Functional Requirements
- Test Driven Development
- User Testing
- Deployment & CI/CD
- Post Deployment Smoke Testing
- Product Validation
- Metrics that Matter

At a lower level, some technical skills you need to learn include:

- Context Engineering
- Prompt Engineering
- Security
- Performance and Optimization
- SEO/GEO (Search Engine Optimization, Generative Engine (AI) Optimization)
- SudoLang Syntax and Best Practices
- Target Language Syntax and Best Practices
- Software Architecture
- Agent Skills
- Agent Orchestration
- Agent Containerization, Sandboxing, Deployment

And of course, a lot more about AI in general.
